AS an admin,I want to use admin-only permissions to manage system data so that i can administer the platform security

Acceptance Criteria:
- admin can enter system using correct username and password
- non-admin user access admin page will return 403 code
- admin can add doctor
- admin can delete doctor
- run a MYSQL stored procedure that returns monthly appointment counts


## Epic

As a user, I want to register and log in to get an appointment and manage my appointment, and get consultation with doctor, so that I can receive care online.

## User Stories (bullets)

- As a user, I can check doctor list without log in
- As a user, I can use the email and password to register to get appointment
- As a user, I can log in website to get appointment
- As a user, I can log in website to manage my appointment
- As a user, I can log out website
- As a user, I can log in and get consultation with doctor
- As a user, I can check nearest appointment
